# Askai
Ask AI and receive answer quick using  commandline

# Setup
    chmod +x askai

    add to bin (to access globally)
    -------------------------------
    sudo cp askai /usr/bin 

# Usage

    askai [question]
    eg:- askai what is color of apple?

    reset api key (if api key is existing)
    --------------------------------------
    askai reset

    show the saved api key (if api key is existing)
    -----------------------------------------------
    askai show 

    setup api key (if api key not stored)
    -------------------------------------
    askai


# Note 

    Default location of apik key stored
    ------------------------------------
    /home/user/.api_keys/askai/.apikey
